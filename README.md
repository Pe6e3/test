# Страница юриста (строгий modern + bokeh)

Собрана статичная страница с:

- современным строгим дизайном и фоновым bokeh-эффектом;
- блоком с изображением специалиста;
- юридическим контентом, который обычно размещают на таких сайтах:
	- политика конфиденциальности;
	- пользовательское соглашение;
	- отказ от ответственности;
	- cookies и технические данные;
	- интеллектуальная собственность;
	- претензионный порядок;
	- контакты и реквизиты.

## Файлы

- `index.html` — структура страницы;
- `styles.css` — визуальный стиль и адаптив;
- `assets/profile-photo.svg` — текущий файл изображения.

## Как заменить фото на ваше

1. Положите ваш файл в `assets/` (например, `assets/profile-photo.jpg`).
2. В `index.html` замените:

```html
<img src="./assets/profile-photo.svg" alt="Портрет специалиста" />
```

на:

```html
<img src="./assets/profile-photo.jpg" alt="Портрет специалиста" />
```
